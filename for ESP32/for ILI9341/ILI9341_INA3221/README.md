## List of Pins
Listed below are the current pins occupied for `Plan Bee`: 
<br/>
ILI9341 Display
<br/>
|ILI9341 Display	|Type of Pin on ESP32 |
| :-------------:	|:--------:		    |
|MISO			|12	 	          |
|LED			|3v3		          |
|SCK			|32	     		    |
|MOSI			|33			    |
|DC			|26	     		    |
|RESET		|25			    |
|CS			|27			    |
|GND			|GND       		    |
|VCC			|3v3	       	    |
<br/>
This is for the 2.4 inch ILI9341 Display.
<br/>
<br/>

<br/>
|DS18B20		|Type of Pin on ESP32 |
